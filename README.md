# 🐾 PetShop - Laravel MVC

## 📌 Descripción
Proyecto web desarrollado en Laravel usando el patrón MVC (Modelo – Vista – Controlador), simulando la gestión de una tienda de mascotas.

## ⚙️ Tecnologías
- Laravel
- PHP
- MySQL
- Blade
- Bootstrap / Tailwind

## 🧱 Estructura
- Modelos: manejo de datos con Eloquent ORM
- Controladores: lógica del sistema
- Vistas: interfaces con Blade

## 🗄️ Base de datos
Se utiliza MySQL con migraciones y tablas relacionadas.

## 🔐 Funcionalidades
- Login y registro de usuarios
- CRUD de productos
- CRUD de categorías
- Gestión de pedidos
- Administración de usuarios
- Perfil y configuración

## 🚀 Ejecución
- composer install
- npm install
- configurar .env
- php artisan migrate
- php artisan serve
